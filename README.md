# Calendar Reminder 📆🥂
## It is little pet project 💻🔨
Using this project 📱, you can easily add notes to remind you of important 📌📅 dates in your life. For example, exam dates 🙋🏻‍♂️, family members' birthdays 👨‍👩‍👦‍👦, wedding dates 👩‍❤️‍👨👰🏻, and more 🌱🎊
